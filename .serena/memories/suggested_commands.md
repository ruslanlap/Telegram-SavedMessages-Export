# Suggested Commands
- Install deps: `pip install -r requirements.txt`
- Show CLI help/examples: `python3 telegram_to_notion.py --help`
- Dry-run export with filters: `python3 telegram_to_notion.py --dry-run --word github --type Text`
- Save matched messages locally (default dir when flag used): `python3 telegram_to_notion.py --save`
- Syntax check: `python3 -m py_compile telegram_to_notion.py`
